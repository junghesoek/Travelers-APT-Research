# Travelers-APT-Research

## Forensic IT Research Project: Spotify Network Analysis

### Purpose
This repository contains a comprehensive forensic analysis of the Spotify user network centered around "The Traveler" (https://open.spotify.com/user/w5j8x1tlo0desiwgo7f0ulpc1).

### Research Focus
- Analysis of music consumption patterns
- Network mapping of artist relationships
- Cultural influence tracking through musical preferences
- Digital footprint analysis through streaming data

### Repository Structure
```
├── README.md                    # This file
├── AGENTS.md                    # Research methodology and agent instructions
└── Spotify-Musicians/           # Artist analysis directory
    ├── [Artist Name]/
    │   ├── SPOTIFY.md          # Spotify-specific artist information
    │   ├── README.md           # General artist information
    │   └── Spotify-Releases/   # Discography analysis
    │       ├── [Release Name]/
    │       │   ├── [Song Name].md  # Song lyrics and analysis
    │       │   └── ...
    │       └── ...
    └── ...
```

### Data Collection Methodology
1. **Artist Profiling**: Complete Spotify and biographical information for each followed artist
2. **Discography Mapping**: Comprehensive release cataloging
3. **Lyric Analysis**: Complete song text collection for content analysis
4. **Network Analysis**: Relationship mapping between artists and genres

### Research Questions
- What musical preferences define "The Traveler's" cultural identity?
- How do artist relationships reveal network patterns?
- What cultural narratives emerge from the collected musical data?
- How can digital streaming behavior inform forensic analysis?

### Compliance Notice
This research is conducted for academic and forensic analysis purposes only. All data collected is publicly available through Spotify's platform and is used solely for research purposes.
